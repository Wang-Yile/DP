# 换根 DP

在第一次 DFS 时预处理一些信息，基于这些信息，在第二次 DFS 时选择合适的根节点进行 DP。

当然，也可以使用答案之间的关系 $O(n)$ 地推出所有其它答案，无需 $n$ 次树形 DP。
